PROYECTO VOTACIONES
Especificacion del cliente (Gobierno):


Task 0.1: Diseñar las interfaces de usuario
Task 0.2: Diseñar el modelo de las bases de datos
Task 0.3: Desplegar las bases de datos

EPICs 

EPIC 1: Gestionar módulo de manejo de candidatos

User Story 1.1: Como candidato me encuentro interesado en poder postularme con las suficientes
garantías electorales.

Task 1.1.1: Construir un servicio web para opinar y evaluar a los candidatos
Task 1.1.2: Construir un servicio web para votar por el candidato de preferencia
Task 1.1.3: Diseñar los mockups correspondientes a las UI de vista de candidatos y votación 
Task 1.1.4: Construir una UI para mostrar las opiniones certificadas, características y evaluación 
            de los candidatos.
Task 1.1.5: Construir una UI para mostrar el resultado de las votaciones.
Task 1.1.6: Diseñar los casos de prueba para la opinión, evaluación y votación de los candidatos.
Task 1.1.7: Llevar a cabo los casos de prueba anteriormente mencionados.
Task 1.1.8: Diseñar los casos de prueba para las UI enunciadas.
Task 1.1.9: Realizar los casos de prueba de las UI.


EPIC 2: Gestionar el módulo de manejo de usuarios

User Story 2.1: Como candidato me interesa que quienes voten sean las personas que deben hacerlo
                una sola vez y pertenezcan al grupo adecuado para hacerlo.

Task 1.1.1: Construir un servicio web para que los votantes puedan loguearse
Task 1.1.2: Construir un servicio web para verificación e ingreso de los usuarios y saber si
            pueden votar
Task 1.1.3: Diseñar los mockups correspondientes a las UI login y ingreso-verificación 
Task 1.1.4: Construir una UI para llevar a cabo el registro de usuarios.
Task 1.1.5: Construir una UI para el ingreso y la verificación de los potenciales votantes.
Task 1.1.6: Diseñar los casos de prueba para el registro, ingreso y verificación de usuarios
Task 1.1.7: Llevar a cabo los casos de prueba anteriormente mencionados
Task 1.1.8: Diseñar los casos de prueba para las UI de login e ingreso-verificación.
Task 1.1.9: Realizar los casos de prueba de las UI de login e ingreso-verificación.
